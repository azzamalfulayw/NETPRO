PowerShell Global Find & Replace Script

This script is basically a "Regex Find & Replace" execution engine that runs directly against your hard drive. It is extremely powerful for fixing systemic typos or globally renaming variables across thousands of files instantly without breaking binaries or compilation folders.

How it Works:

  • Get-ChildItem -Recurse -File -Filter "*.cs" dynamically searches for every C# file in the source directory.
  • Where-Object explicitly filters out locked/build directories like bin, obj, and Migrations so we don't accidentally corrupt executable builds or Database History.
  • Get-Content -Raw loads the entire file into memory as a singular string.
  • -replace executes a fast string replacement for target words.
  • Set-Content -NoNewline writes the patched string directly back into the hard drive transparently, overriding the old content.

The Script:

$files = Get-ChildItem -Path ".\api" -Recurse -File -Filter "*.cs" | Where-Object { $_.DirectoryName -notmatch "Migrations" -and $_.DirectoryName -notmatch "obj" -and $_.DirectoryName -notmatch "bin" }
foreach ($file in $files) {
    $text = Get-Content $file.FullName -Raw
    $modified = $false
    
    # Add your target words here
    if ($text -match "OldSpelling") {
        $text = $text -replace "OldSpelling","NewSpelling"
        $modified = $true
    }
    
    if ($modified) {
        Set-Content -Path $file.FullName -Value $text -NoNewline
    }
}

How to Run in the Future:

Save the snippet above into a file named replace.ps1 at the root of your project, alter the strings to what you need, and then execute .\replace.ps1 inside a PowerShell terminal instance.
